BlobPad
=======

BlobPad is a note taking application build on top of [BlobStash](https://github.com/tsileo/blobstash).

## Features

- Markdown only notes
- Full notes history

## Based upon

- [BlobStash](https://github.com/tsileo/blobstash)
- [Ace Editor](http://ace.c9.io/)

## Roadmap

- Elastic search based search engine
- PDF support (+ OCR)
